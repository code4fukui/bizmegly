# bizmegly

大阪府堺市のオープンデータから企業の写真を閲覧できるWebアプリケーションです。

## デモ
[大阪堺市の企業フォト巡り - さかしる企業オープンデータの写真から始まる出会い](https://code4fukui.github.io/bizmegly/sakai.html)

## 機能
- 堺市のオープンデータから企業の写真をグリッド形式で表示
- 写真をクリックすると、企業の詳細情報が表示される
- 企業のWebサイトやGoogleマップの所在地へのリンクを提供

## データ / API
このプロジェクトは、[さかしる](https://sakacil.com/)が提供する[さかしる企業オープンデータ](https://github.com/codeforosaka/sakai-biz-opendata/)の企業画像データを使用しています。
